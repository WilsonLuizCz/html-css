Oque é Semântica?

A Semantica é a ciência que estuda o significado dos elementos e define que o significado é mais importante que as formas. Antigamente no HTML 4 se usava algumas tags de "formas" no HTML que hoje estão obsoletas no site da W3C, ou seja,não estão mais indicadas para uso pois elas funcionam mais a qualquer momento serão banidas, exemplo:tag marquee, e tag "U" .

Hoje o HTML5 é exclusivo para tags de significado deixando todas as tags de forma com as Css.
